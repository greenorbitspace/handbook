# unicorn-magic

> Some useful utilities I often need

*I'm not accepting requests.*

## Install

```sh
npm install unicorn-magic
```

## Usage

```js
import {delay} from 'unicorn-magic';

await delay({seconds: 1});

console.log('1 second later');
```
